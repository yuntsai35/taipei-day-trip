### Main Features:

- Register, Log in, Sign up
- Explore all attractions on homepage
- Search keywords to explore desired attractions
- Responsive Web Design, fit every platform
- Allow lazy loading
- Custom image carousel

### Backend Techniques:

Web Framework: RESTful APIs with Python FastAPI 
Database: MySQL with connection pooling
Authenticating users with JWT
Third-party payment service with Tappay SDK
Deploy website on AWS EC2 Linux instance.
Constructed backend code using MVC design pattern


### Frontend Techniques:

- `HTML`
- `Javascipt`
- `CSS`
- `RDW`
